# TMAA-Projekt

Jdnoduchá hra kde je za úkol uhodnout minecraft item podle jeho receptu do určitého počtu pokusů.

Něco na způsob wordle.

item se nachází v receptu na tomto místě = zelená, item se nachází jinde = žlutá


Herní režimy:

Náhodný předmět,

Uhodnout co nejvícce v časovém limitu,

Denní úkol

Výzvy (uhodnout za co nejméně pokusů)


Funkcionality:

Denní úkoly,

Série (úspěšná série, denní série),

Upozornění (denní úkol),

Statistiky - ukládání (lokálně/cloud), výpis statistik
